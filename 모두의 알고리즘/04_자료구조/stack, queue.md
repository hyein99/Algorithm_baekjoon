# Queue
### 1. queue 개념
first in first out
* enqueue: 삽입
* dequeue: 삭제

### 2. 리스트로 queue 구현
```python
qu = []     # 초기화
qu.append() # enqueue
qu.pop(0)   # dequeue
```
<br>

# Stack
### 1. stack 개념
last in first out
* push: 삽입
* pop: 삭제

### 2. 리스트로 stack 구현
```python
st = []      # 초기화
st.append()  # push
st.pop()     # pop
```
